# 风险管理系统

这是一个用于风险评分和管理的Python项目。

## 项目结构

- `example.py`: 示例代码
- `scorecard_model.py`: 评分卡模型实现
- `requirements.txt`: 项目依赖

## 安装

1. 克隆仓库
2. 创建虚拟环境: `python -m venv .venv`
3. 激活虚拟环境: `source .venv/bin/activate`
4. 安装依赖: `pip install -r requirements.txt`

## 使用

运行示例代码:

```bash
python example.py
```
